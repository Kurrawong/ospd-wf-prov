# OSPD Workflow Provenance

This repository contains Semantic Web resources created during the [Open Geospatial Consortium (OGC)](https://www.ogc.org)'s [Open Science Persistent Demonstrator 2025](https://www.ogc.org/initiatives/open-science/) project, bundled according to the [Prez Manifest](https://prez.dev/Manifest) model.

These resources are destined for use in a node of the OGC's [Rainbow reference data platform](https://defs.opengis.net) where they demonstrate the display and use of standardised provenance information for scientific workflows.
